# KOL Insight Agent（KOL访谈洞察智能体）

> 本智能体用于系统化分析主任医师访谈内容，自动生成产品临床定位、临床路径与患者就医路径策略报告。

## 🧠 智能体定位
- **角色**：资深医学策略顾问（Market Access & HEOR）
- **输入**：KOL访谈转录稿 / 核心观点汇总
- **输出**：结构化医学洞察报告（Markdown）

## 📥 使用方法（3步）

### Step 1：准备输入
将访谈核心观点整理为纯文本，放入 `example_input.md` 作为参考，或直接复制以下内容：

### Step 2：调用 Prompt
复制 `prompt.txt` 中的完整 Prompt，将上面的输入内容粘贴到 Prompt 中的 `【在此处粘贴访谈转录稿，或总结后的核心观点】` 位置。

### Step 3：运行智能体
将组合后的 Prompt 提交给任意大模型（推荐：GPT-4o / Claude 3.5 / DeepSeek-Chat），即可获得分析报告。

## 📂 文件说明
- `prompt.txt`：智能体核心 Prompt（不可随意修改）
- `example_input.md`：示例输入（可替换）
- `example_output.md`：示例输出（供参考格式）

## ⚠️ 注意事项
1. 单次输入建议不超过 15,000 字（约 3–5 篇访谈稿精华）
2. 输出需由医学经理复核，智能体仅提供初稿
3. 禁止用于直接生成临床决策或监管递交文件

## 📈 后续优化方向

- 与 CRF Generator Agent 联动，形成“临床洞察 → CRF设计”闭环
